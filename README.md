**Stack Size Adjuster** lets you dynamically customize the maximum stack sizes of item categories. Want ender pearls to stack to 64? Want potions and swords to stack to 16? Want blocks to be non-stackable? This mod makes it possible.

Part of the **Instant Gratification Collection** — mods that respect the player's time.

---

## ✨ Features

- **Three Category Configuration**:
  - **Category 1 (Normal 64-stack items)**: Blocks, raw food, ores, etc.
  - **Category 2 (Normal 16-stack items)**: Snowballs, eggs, ender pearls.
  - **Category 3 (Normal 1-stack items)**: Weapons, tools, armor, potions, stews.
- **Dynamic Limit Scaling**: Change any category's limit to **64**, **16**, or **1** at runtime.
- **Modded Item Compatibility**: Works out of the box with modded items.
- **Optional GUI integration**: Integrates with ModMenu and YetAnotherConfigLib (YACL) for easy in-game configuration. Works perfectly without them using GameRules.

---

## 📋 Configuration

You can configure options in-game using `/gamerule` or via the in-game GUI:
- `stack-size-adjuster:items_64_limit` (Default: 64)
- `stack-size-adjuster:items_16_limit` (Default: 16)
- `stack-size-adjuster:items_1_limit` (Default: 1)

## 📦 Install

1. Install **[Fabric API](https://modrinth.com/mod/fabric-api)**.
2. Download the mod jar and place it in your `mods` folder.
3. Launch the game.
